# Generative AI Learning Guide

A comprehensive learning resource covering the fundamentals of Generative AI, Large Language Models (LLMs), and Prompt Engineering techniques.

---

## 📚 Workspace Structure

### 1. **01_Intro_Gen_AI** - Introduction to Generative AI
An introductory guide to understanding the core concepts of Generative AI and Large Language Models.

**Topics Covered:**
- **What is Generative AI?** - Understanding AI systems that create new content
- **Large Language Models (LLMs)** - Advanced AI systems for text understanding and generation
- **How LLMs Work** - Token-based prediction and pattern recognition mechanisms
- **Models and Capabilities** - Exploring different AI models (ChatGPT, Claude, Gemini, Grok)
  - General-purpose LLMs
  - Reasoning models
- **Tokens** - The smallest units of text that models use
- **Context & Context Window** - Information capacity of LLMs
- **Inference** - Generating outputs from trained models

### 2. **02_Prompt_Engineering** - Prompt Engineering Techniques
Master the art of designing effective prompts to get the best results from LLMs.

**Topics Covered:**
- **Prompt Engineering Fundamentals** - Structuring inputs for clarity and accuracy
- **Hallucination** - Understanding false information generation in AI
- **Types of Prompting:**
  - Zero-shot Prompting - Instructions without examples
  - Few-shot Prompting - Learning from provided examples
  - Chain-of-thought Prompting - Step-by-step reasoning techniques

### 3. **03_LLM** - Large Language Models Deep Dive
Explore the practical implementation and utilization of Large Language Models.

**Topics Covered:**
- **LLM Architectures** - Understanding how modern LLMs are built
- **Model Training & Fine-tuning** - Adapting LLMs for specific tasks
- **Working with APIs** - Integrating LLMs into applications
- **Practical Applications** - Real-world use cases and examples
- **Node.js Integration** - Building applications with JavaScript/Node.js

### 4. **04_Projects** - Hands-On Projects
Apply your knowledge through real-world projects combining concepts from all sections.

**Project Focus:**
- **Generative AI Applications** - Building practical systems with LLMs
- **End-to-End Integration** - Combining prompt engineering with API usage
- **Production-Ready Code** - Best practices for deploying AI solutions

---

## 🎯 Learning Objectives

By working through this material, you will understand:
- The fundamentals of Generative AI and how modern language models function
- The role of tokenization and context windows in AI systems
- Practical techniques for crafting effective prompts
- Different prompting strategies and when to use them
- How to avoid common pitfalls like AI hallucinations

1. Start with **01_Intro_Gen_AI** to build foundational knowledge
2. Progress to **02_Prompt_Engineering** to learn practical techniques

## 💡 Key Concepts at a Glance

| Concept | Description |
|---------|-------------|
| **GenAI** | AI systems that generate new content (text, images, code, etc.) |
| **LLM** | Large Language Models - AI systems trained on massive text data |
| **Token** | Smallest unit of text (word, subword, or character) |
| **Context Window** | Maximum tokens an LLM can process at once |
| **Inference** | Process of generating output from a trained model |
| **Prompt Engineering** | Art of crafting inputs for optimal AI outputs |
